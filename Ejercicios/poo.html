Programacion orientada a objetos

Clases y objetos

  • Clases: Una clase es una plantilla para crear objetos. Define un tipo de objeto según sus propiedades (atributos) y métodos (funciones)
  • Objetos: Una instancia de una clase. Es una entidad que tiene un estado y comportamiento definidos por sus propiedades y métodos.

Clases y Objetos

Herencias

  • Herencia: Es un mecanismo que permite crear una nueva clase basada en una clase existente. La clase nueva hereda las propiedades y métodos de la clase base.

Ejemplo de herencia

Polimorfismo

  • Polimorfismo: Permite que diferentes clases se traten como si fueran de una clase común. Es la capacidad de un método de hacer cosas diferentes en función del objeto que lo llama.
